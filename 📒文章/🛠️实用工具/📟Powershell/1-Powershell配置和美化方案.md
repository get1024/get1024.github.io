# Powershellй…ҚзҪ®е’ҢзҫҺеҢ–ж–№жЎҲ

## еүҚиЁҖ

`Terminal` е’Ң `е‘Ҫд»ӨиЎҢе·Ҙе…·` зӣёдҝЎеӨ§е®¶йғҪдёҚйҷҢз”ҹпјҢдҪҶжҳҜдҪңдёәи®Ўз®—жңәзӣёе…ідё“дёҡзҡ„еӯҰз”ҹ/з§‘з ”дәәе‘ҳ/е·ҘдҪңдәәе‘ҳпјҢдёҖе®ҡдәҶи§ЈеҲ° `Powershell`гҖӮ

дҪҶжҳҜеҺҹз”ҹ `Powershell` еңЁиҜӯжі•й«ҳдә®гҖҒд»Јз ҒиЎҘе…ЁгҖҒж ·ејҸзҫҺеҢ–ж–№йқўе№¶дёҚиғҪе®Ңе…Ёж»Ўи¶іеӨ§е®¶дёӘжҖ§еҢ–зҡ„йңҖжұӮпјҢиҝҷйҮҢдҫҝжҸҗеҮәдәҶ `Powershell` й…ҚзҪ®е’ҢзҫҺеҢ–ж–№жЎҲгҖӮ

е…ҲзңӢй…ҚзҪ®зҫҺеҢ–е®ҢжҲҗзҡ„ж•Ҳжһң

- Windows з»Ҳз«ҜзӘ—еҸЈпјҲжңүиғҢжҷҜжҳҜеӣ дёәз»Ҳз«ҜзӘ—еҸЈйҖҸжҳҺпјҢеҗҺйқўжҳҜжҲ‘зҡ„жЎҢйқў**еЈҒзәё**пјү
![](assets/1-powershellй…ҚзҪ®е’ҢзҫҺеҢ–ж–№жЎҲ/з»Ҳз«Ҝж ·ејҸ.png)
- VSCode иҮӘеёҰз»Ҳз«Ҝ
![](assets/1-powershellй…ҚзҪ®е’ҢзҫҺеҢ–ж–№жЎҲ/vscodeз»Ҳз«Ҝж ·ејҸ.png)
жҳҜдёҚжҳҜжҜ”еҺҹз”ҹзҡ„еҘҪзңӢе‘ўпјҹжҺҘдёӢжқҘзңӢзңӢе…·дҪ“еҰӮдҪ•й…ҚзҪ®еҗ§пјҒеҰӮжһңдҪ еҸӘжҳҜжғіеӨҚеҲ»жҲ‘зҡ„ж ·ејҸпјҢиҜ·зӣҙжҺҘи·іиҪ¬иҝҷйҮҢпјҡ [еӨҚеҲ»жҲ‘зҡ„й…ҚзҪ®](#еӨҚеҲ»жҲ‘зҡ„й…ҚзҪ®)

## е®үиЈ… Powershell 7

еңЁ `Windows Terminal` (еҜ№дәҺ Windows 11 з”ЁжҲ·) <span class="marker-evy">жҲ–</span> `Windows PowerShell` (еҜ№дәҺ Windows 10 з”ЁжҲ·) дёӯпјҢеӨҚеҲ¶д»ҘдёӢе‘Ҫд»ӨиҝҗиЎҢгҖӮ

```sh [powershell]
winget install --id Microsoft.PowerShell --source winget
```

иҝҷж ·й»ҳи®ӨжҳҜе®үиЈ…еңЁ `C:\` зӣҳдёӯпјҢжҲ‘д»¬д№ҹжҺЁиҚҗеҰӮжӯӨгҖӮеҰӮжһңдҪ е®һеңЁдёҚжғіиҝҷд№ҲеҒҡпјҢеҸҜд»ҘйҖҡиҝҮиҝҪеҠ еҸӮж•° `-i` жү“ејҖдәӨдә’ејҸе®үиЈ…гҖӮ

иҖҗеҝғзӯүеҫ…еҗҺжҳҫзӨәе®үиЈ…жҲҗеҠҹгҖӮ

![](assets/1-powershellй…ҚзҪ®е’ҢзҫҺеҢ–ж–№жЎҲ/powershell7е®үиЈ….mp4)

еҰӮжһңдҪ жңүжӣҙеӨҡйңҖжұӮпјҢиҜ·жҹҘзңӢе®ҳж–№ж–ҮжЎЈпјҡ[еңЁ Windows дёҠе®үиЈ… PowerShell](https://learn.microsoft.com/zh-cn/powershell/scripting/install/installing-powershell-on-windows?view=powershell-7.4#install-powershell-using-winget-recommended)

## й…ҚзҪ®

### и®ҫзҪ®й»ҳи®Ө

жү“ејҖ `PowerShell7` и®ҫзҪ®йЎөпјҢйҖүжӢ©й»ҳи®Өжү“ејҖдёә `PowerShell`

![](assets/1-powershellй…ҚзҪ®е’ҢзҫҺеҢ–ж–№жЎҲ/powershell%20setting.png)

### е®үиЈ… `Posh-Git`

`PowerShell7` иҝҗиЎҢд»ҘдёӢе‘Ҫд»Ө

```shell [powershell]
Install-Module -Name posh-git -Force
```

### е®үиЈ… `Oh-my-posh`

дҝЎд»» `PSGallery` еә“

```sh [powershell]
Set-PSRepository -Name PSGallery -InstallationPolicy Trusted
```

еңЁ `Powershell` дёӯиҝҗиЎҢд»ҘдёӢе‘Ҫд»ӨпјҢиҖҗеҝғзӯүеҫ…е®үиЈ…жҲҗеҠҹ

```sh [powershell]
winget install JanDeDobbeleer.OhMyPosh -s winget
```

### е®үиЈ…еӯ—дҪ“

```sh [powershell]
oh-my-posh font install meslo
```

е®үиЈ…еӯ—дҪ“еҗҺпјҢжӮЁйңҖиҰҒй…ҚзҪ® `Windows з»Ҳз«Ҝ` жүҚиғҪдҪҝз”Ёе®ғгҖӮиҝҷеҸҜд»ҘйҖҡиҝҮдҝ®ж”№ Windows з»Ҳз«Ҝи®ҫзҪ®иҪ»жқҫе®ҢжҲҗпјҲй»ҳи®Өеҝ«жҚ·ж–№ејҸпјҡ `CTRL + SHIFT + ,` пјүгҖӮеңЁжӮЁзҡ„ `settings.json` ж–Үд»¶дёӯпјҢеңЁ `profiles` дёӯзҡ„ `defaults` еұһжҖ§дёӢж·»еҠ В `font.face` еұһжҖ§пјҡ

иҝҷйҮҢжҸҗдҫӣеҮ дёӘе…ій”®ж–Үд»¶пјҢеҸӘиҰҒдҪ й…ҚзҪ®е®Ңе…ЁдәҶдёҖдёӢдёүдёӘж–Үд»¶пјҢйӮЈд№Ҳе°ұиғҪеҫ—еҲ°жҲ‘зҡ„ж ·ејҸж•ҲжһңдәҶгҖӮ
